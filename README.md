# code-cooker
Simple data analysis software like Advanced Data Analysis(Code Interpreter).

## Setup

```sh
$ git clone https://github.com/karaage0703/code-cooker
$ cd code-cooker
$ pip install -e .
```

Edit `./cooker/.config`


## Usage

```sh
$ python3 app.py 
```

## Test

```sh
$ python3 tests/test_cooker.py
```
